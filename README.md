# Gestor de Tareas - Semana 16

Aplicación de escritorio modular con interfaz gráfica y atajos de teclado, evolucionando la Tarea de la Semana 15.

## Atajos de Teclado
* **Enter**: Añadir tarea.
* **Tecla C**: Marcar como completada.
* **Tecla D**: Eliminar tarea.
* **Escape**: Salir del programa.

## Estructura del Proyecto
* `main.py`: Punto de entrada.
* `modelos/`: Definición de datos.
* `servicios/`: Lógica de la aplicación.
* `ui/`: Interfaz y eventos.
